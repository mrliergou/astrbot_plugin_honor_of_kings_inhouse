- 对局记录管理

✅ **管理员权限**

## 部署说明

### 安装步骤

1. **复制插件目录**
   ```bash
   cp -r astrbot_plugin_honor_of_kings_inhouse /path/to/astrbot/plugins/
   ```

2. **重启 AstrBot**
   ```bash
   # 重启 AstrBot 服务
   ```

3. **测试插件**
   ```
   在群聊中发送：/报名列表
   ```

### 配置管理员

编辑 `data/hok_inhouse_state.json`，在对应群组的 `config.admin_users` 中添加用户ID：

```json
{
  "chats": {
    "群组ID": {
      "config": {
        "admin_users": ["用户ID1", "用户ID2"]
      }
    }
  }
}
```
